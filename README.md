# userScripts

Some quick and dirty userscripts


## imdbLetterboxedLink.js

Straightforward links for IMDB and a reddit search (truefilm and criterion) via Google

## imdbAdditionalButtons.js

This is the same as the previous but adds the ability to post the movie (IMDB ID, Title, and Year) to a Google Form. 

1. create a form with three short answer text boxes
2. Top left -- three dot menu > prefill form
3. fill out the form and get the link

  https://docs.google.com/forms/d/e/7tZqF4amN2VyEwpXxJBUKohGdL19RcTMeQvHs0f8bgC/viewform?usp=pp_url&entry.1231231231=tt0068646&entry.4564564564=The+Godfather&entry.7897897897=1972

  In the script, use

  * `1231231231`
  * `4564564564`
  * `7897897897`
  * `7tZqF4amN2VyEwpXxJBUKohGdL19RcTMeQvHs0f8bgC` is the form ID
